
## Getting Started

### Prerequisites
* Install [OCaml](https://ocaml.org/docs/installing-ocaml)
* Install [Dune](https://dune.build/install)

### Build
```
dune build
```
or
```
dune build --watch
```

### Auto-format
```
dune fmt
```

### Build and run self-check
```
dune exec tree_calculus_reference_implementation
```
